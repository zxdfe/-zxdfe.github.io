  1. 安装vue2.7相关的插件，vue2和vite搭配使用必须安装
js
pnpm i @vitejs/plugin-vue2 -D  # 开发环境使用
  1. 新建一个vite.config.js，放入初始配置
js
// vite.config.js
import vue from '@vitejs/plugin-vue2'

export default {
  plugins: [vue()]
}
  1. 安装Vue2.7.14
js
 pnpm i vue@^2 
或者 
pnpm i vue@2.7.14
  1. 新建src文件夹，移入main.js,(删除原来的)
js
import Vue from 'vue'
import App from './App.vue'

Vue.config.productionTip = false

new Vue({
  render: h => h(App),
}).$mount('#app')
  1. 新建App.vue
js
写入一些自己的代码，
  1. index.html中 src main.js路径修改一下
js
# 在index.html中修改 main.js的引入路径
<script type="module" src="/src/main.js"></script>

# 注意type="module"   ESM的模块化方式！！！
  1. pnpm dev启动项目
