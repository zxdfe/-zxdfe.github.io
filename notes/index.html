js
修改元素的样式 有哪些方式？
1. style    ==> 如果要修改的样式比较少的时候用
2. className   ==> 缺陷： 会覆盖原本自身的类名
3. classList   ==> 当需要修改的样式比较多的时候，推荐使用

! 注意 不需要加点 .

定时器 #

js
setInterval(fn, wait)  
// 第一个参数 要执行的函数
// 第二个参数 间隔时间  wait 单位 ms 毫秒
==> 每隔一段时间，就去调用执行fn这个函数，只要我们不关闭定时器，就不会停止

回调函数 ： 咱们第一个参数 fn 也叫做回调函数 
回头再调用的函数 ， 一开始不执行， 间隔时间到了，再回去调用执行的函数
